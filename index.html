Terminal

Commands to navigate with the terminal

  • pwd : to find out in which directory you are
  • ls : to list the content of the directory you are in
  • cd : to change to another directory
  • cd .. : to go up one directory
Git

Commands in Git

  • git init : To initialize a git repository in a new or an existing project
  • git clone : To copy a git repository
  • git add --all : To add all the changes to be committed
  • git commit -m : To commit the files that have been added, creates a new revision with a log
  • git push : To push your changes to the remote directory
  • git pull : To take the files from the remote directory and merge them with the local one
  • git log : To show a list of commits on a branch with the details
  • git add -A stages All
  • git add . stages new and modified, without deleted
  • git add -u stages modified and deleted, without new
